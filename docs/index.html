30 Integrated Modules

From basic DNS lookups to advanced SSRF probing and JWT exploitation — all in one authenticated web UI.

Quick Start
Up in 60 seconds

Three ways to run CyberScan Pro.

🐍 Python
🐳 Docker
🪟 Windows
# Clone the repository
git clone https://github.com/anshul2414/cyberscan-pro.git
cd cyberscan-pro

# Install dependencies
pip install -r requirements.txt

# Launch
python app.py

# Open → http://localhost:5000
# Login → admin / admin123
# Clone & deploy with Docker Compose
git clone https://github.com/anshul2414/cyberscan-pro.git
cd cyberscan-pro

# Edit docker-compose.yml to set your credentials
# AUTH_USERNAME, AUTH_PASSWORD, SECRET_KEY

docker-compose up -d

# Open → http://localhost:8080
# Clone the repository
git clone https://github.com/anshul2414/cyberscan-pro.git
cd cyberscan-pro

# Double-click or run:
run.bat

# Open → http://localhost:5000
